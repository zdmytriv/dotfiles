### More

Two aliases can created like
```
alias ebt="dotfiles/scripts/aws/ebt.py"
alias ec2="dotfiles/scripts/aws/ec2.py"
```
So these scripts can be used like this from anywhere
```
ec2 profile-dev

# and

ebt profile-dev
```
